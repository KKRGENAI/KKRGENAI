<h2 align="center">🔥 Our Projects</h2>

### 🔍 [Langchain_RAG](https://github.com/KKRGENAI/Langchain_RAG)

Enterprise-grade **Retrieval-Augmented Generation** system built for real-world document intelligence.

- **Stack:** LangChain · Pinecone · Streamlit · Ollama · LLaMA models · nomic-embed-text
- **Features:** Incremental PDF ingestion, semantic search, source attribution, modular architecture
- **Use Cases:** Enterprise document Q&A, internal knowledge platforms, policy/documentation exploration
- **Roadmap:** FastAPI layer, Docker containerization, CI guardrails, document versioning

<hr style="border:0.01px solid blue">

### 🌿 [VagbhataAI](https://github.com/KKRGENAI/VagbhataAI)

An intelligent **RAG system for Ayurvedic knowledge**, grounded in classical Sanskrit texts.

- **Stack:** Streamlit · Google Gemini (gemini-2.5-flash-lite) · LangChain · LangGraph · Pinecone · PostgreSQL · Google text-embedding-004
- **Features:** Source attribution (Sutra references), safety/medical disclaimers, persistent conversation memory, agentic state machine
- **Purpose:** Bridging ancient Ayurvedic wisdom with modern AI interfaces

<hr style="border:0.01px solid blue">

### 💬 [ai-chatbot-level1](https://github.com/KKRGENAI/ai-chatbot-level1)

**Level 1 of our AI Learning Series** — an AI Chatbot with conversation memory built for education.

- **Stack:** Python · OpenAI GPT models
- **Features:** Natural language understanding, conversation memory, clean modular code
- **Purpose:** Teaches foundational AI concepts; Level 2 adds Streamlit UI + streaming output

<hr style="border:0.01px solid blue">

### 📈 [financial-ai-agent](https://github.com/KKRGENAI/financial-ai-agent)

An **educational Financial AI Agent** powered by Claude API with real-time market data.

- **Stack:** Python 3.10+ · Anthropic Claude API · FastAPI · yfinance · Pydantic · pytest
- **Features:** Real-time stock prices, historical OHLCV data, multi-stock comparison, financial news, market indices
- **Interfaces:** REST API + CLI
- **Tools:** `get_stock_price` · `get_stock_history` · `compare_stocks` · `get_market_news` · `get_economic_indicators`

<hr style="border:0.01px solid blue">
